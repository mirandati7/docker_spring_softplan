Instalação
------------
Front end SOFTPLAN 

#### Download:

Download from [Github releases](https://github.com/mirandati7/front-softplan).

#### Using The Command Line:

__Via Git__
- Clone to your machine
```
git clone https://github.com/mirandati7/front-softplan
```

__Via NPM__
```bash
npm install 
```
